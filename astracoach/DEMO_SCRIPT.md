## RECORDING CHECKLIST

### Before Recording
- [ ] Backend running locally with Firestore connected
- [ ] Demo data seeded (dashboard should show all populated tabs)
- [ ] Screen Studio configured: 1920x1080, 60fps, dark desktop wallpaper
- [ ] Eleven Labs voice ready for voiceover sections
- [ ] Browser in full-screen Chrome, no bookmarks bar, dark theme
- [ ] Close all other apps/notifications
- [ ] Mic tested (use a good condenser mic, not laptop mic)

### Screen Studio Tips
- Use **zoom-in** on mouse clicks for tool pills and badges
- Use **slow zoom** on the avatar during speaking state
- Add **subtle cursor highlight** so judges can track what you're clicking
- Export at **1080p minimum**, preferably 4K downscaled to 1080p for crisp text

### Eleven Labs Tips
- Use a natural male voice for voiceover (not the robotic preset)
- Record voiceover sections SEPARATELY from live Astra interaction
- Mix voiceover at -3dB relative to Astra's voice so the AI voice stands out
- Add 0.5s silence gaps between voiceover and live interaction transitions

### Editing Flow
1. Record all live Astra interactions FIRST (Screen Studio)
2. Record voiceover narration separately (Eleven Labs or direct mic)
3. Layer voiceover on top in your video editor
4. Add the opening text animation (After Effects, Canva, or CapCut)
5. Add architecture diagram slide (static image, 30 seconds)
6. Add closing card
7. Add background music (low, ambient)
8. Final mix: ensure Astra's voice is loudest, voiceover second, music quiet
